# 🌎 Interactive South America Map

Aplicación web interactiva que permite explorar países de Sudamérica mediante un mapa SVG dinámico, 
mostrando información detallada, tooltips, 
panel lateral y soporte multilenguaje (Español / Ingles).

## ✨ Características

- 🗺️ Mapa SVG interactivo de Sudamérica
- 📍 Tooltip con bandera y nombre del país
- 📊 Panel lateral con información detallada
- 🌐 Soporte multilenguaje (Español / Inglés)
- 🎨 Modo claro / oscuro
- 🧭 Coloreado por región o por país
- ⚡ Animaciones con Framer Motion
- 💅 Estilos con Tailwind CSS

---

## 🛠️ Tecnologías utilizadas

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Context API**
- **SVG**

---

## 🌐 Internacionalización (i18n)

La aplicación utiliza Context API para manejar el idioma globalmente:
Idiomas soportados: Español e Inglés
Traducción de:

- Textos de UI
- Etiquetas
- Datos de países
